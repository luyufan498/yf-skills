# 模拟盘交易系统使用指南

## 概述

模拟盘交易系统是一个用于模拟股票交易的脚本工具，支持 A股和港股的交易操作，每个股票有独立的资金池管理。

**重要说明**：
- Agent 在执行时应根据 skill 的实际安装位置，自动替换为完整的绝对路径
- 典型路径格式：`<skill根目录>/scripts/paper_trading.py`

## 交易原则与策略

### 🎯 核心交易纪律

1. **优先基于分析报告操作**
   - 所有交易决策应基于每日分析报告的投资建议
   - 不得在无分析报告支持的情况下进行交易
   - 交易备注中应关联对应的分析报告（如参考报告日期）

2. **严格执行止损止盈**
   - 买入前设定明确的止损价和止盈价
   - 达到止损线及时止损，避免情绪化加仓摊平
   - 达到止盈线分批止盈，保留上涨空间

3. **仓位控制原则**
   - 单只股票初始建仓不超过总资金的 30%
   - 空仓时首次建仓仓位控制在总资金的 10-20%
   - 重仓股（占比 > 50%）应谨慎加仓

4. **分批买卖原则**
   - 大额资金建议分 2-3 批次逐步建仓
   - 上涨趋势中突破关键位可加仓
   - 下跌趋势中跌破支撑位减仓或止损

5. **避免频繁交易**
   - 每日交易次数不超过 2 次（同一股票）
   - 避免日内短线操作（模拟盘适合中长线策略）
   - 相同方向交易间隔至少 1 个交易日

### 📊 基本交易策略

1. **趋势跟随策略**
   - 上涨趋势：突破关键阻力位可加仓
   - 下跌趋势：跌破重要支撑位减仓或止损
   - 趋势不明：观察等待，轻仓试探

2. **价值投资策略**
   - 分析报告给出买入建议且价格安全边际大
   - 价格低于目标价 15-20% 时分批建仓
   - 长期持有，忽略短期波动

3. **催化剂事件策略**
   - 涉及重大利好事件（业绩超预期、政策支持等）及时建仓
   - 事件落地后评估是否继续持有
   - 避免追高，等待回调机会

4. **技术面结合策略**
   - 技术面突破确认（成交量放大）后入场
   - 避免在技术形态走坏时加仓
   - 结合 K 线形态判断买卖点

### ⚠️ 禁止行为

1. **禁止盲目追涨杀跌**
   - 不因短期快速上涨盲目追高
   - 不因短期快速下跌恐慌杀跌
   - 以分析报告为依据，避免情绪化操作

2. **禁止频繁操作**
   - 一日内多次买卖同一股票
   - 频繁改变交易方向
   - 无交易逻辑的随机操作

3. **禁止满仓梭哈**
   - 单只股票持仓不超过总资金 50%
   - 保留一定现金应对机会
   - 避免一次建仓占用全部资金

4. **禁止摊平死扛**
   - 亏损达到止损线严格执行止损
   - 不在亏损时增加仓位摊低成本
   - 及时止损，保存本金

### 📋 交易决策流程

```
1. 生成每日分析报告
   ↓
2. 评估投资建议（买入/卖出/持有）
   ↓
3. 检查当前持仓状态
   ↓
4. 评估资金使用率
   ↓
5. 制定具体交易计划
   - 数量/金额
   - 分批策略
   - 止损/止盈位
   ↓
6. 执行交易（添加详细备注）
   ↓
7. 记录并总结
```

## 数据存储

交易数据存储在 `intermediate/<股票名>/模拟买卖/` 目录下：

```
intermediate/
└── 中科曙光/
    └── 模拟买卖/
        ├── holdings.json     # 当前持仓、资金池状态
        └── operations.json   # 历史操作记录
```

## 核心命令

### 1. 初始化资金池

为指定股票创建独立的资金池。

```bash
python3 <skill根目录>/scripts/paper_trading.py init "股票名称" --capital 500000
```

**参数说明**：
- `股票名称`：必需，如 "赛力斯"、"中科曙光"
- `--capital/-c`：必需，初始资金金额
- `--code`：可选，股票代码（如 sh600000），不提供时自动查询
- `--force/-f`：可选，强制删除现有资金池并重新初始化

**示例**：
```bash
# 正常初始化
python3 <skill根目录>/scripts/paper_trading.py init "赛力斯" --capital 500000

# 强制重新初始化（删除现有数据）
python3 <skill根目录>/scripts/paper_trading.py init "赛力斯" --capital 500000 --force
python3 <skill根目录>/scripts/paper_trading.py init "赛力斯" --capital 500000 -f  # 简写

# 指定股票代码
python3 <skill根目录>/scripts/paper_trading.py init "腾讯控股" --capital 500000 --code hk00700
```

### 3. 买入股票

支持按股数或按金额买入。

```bash
# 按股数买入
python3 <skill根目录>/scripts/paper_trading.py buy "股票名称" --qty 100

# 按金额买入
python3 <skill根目录>/scripts/paper_trading.py buy "股票名称" --amount 10000

# 添加备注
python3 <skill根目录>/scripts/paper_trading.py buy "股票名称" --qty 200 --note "初始建仓"
```

**参数说明**：
- `--qty/-q`：买入股数（与 `--amount` 二选一）
- `--amount/-a`：买入金额（与 `--qty` 二选一）
- `--note/-n`：可选，交易备注

**交易流程**：
1. 自动获取实时价格
2. 验证资金池余额是否充足
3. 资金不足时拒绝交易并提示缺口金额
4. 扣除资金，更新持仓，记录操作

**示例**：
```bash
# 买入 100 股
python3 <skill根目录>/scripts/paper_trading.py buy "赛力斯" --qty 100

# 买入约 10000 元的股票（按金额计算）
python3 <skill根目录>/scripts/paper_trading.py buy "赛力斯" --amount 10000

# 带备注
python3 <skill根目录>/scripts/paper_trading.py buy "中科曙光" --qty 200 --note "技术面加仓"
```

---

### 4. 卖出股票

支持部分卖出或全部卖出。

```bash
# 按股数卖出
python3 <skill根目录>/scripts/paper_trading.py sell "股票名称" --qty 50

# 全仓卖出
python3 <skill根目录>/scripts/paper_trading.py sell "股票名称" --all

# 添加备注
python3 <skill根目录>/scripts/paper_trading.py sell "股票名称" --qty 100 --note "部分止盈"
```

**参数说明**：
- `--qty/-q`：卖出股数（与 `--all` 二选一）
- `--all`：全部卖出（与 `--qty` 二选一）
- `--note/-n`：可选，交易备注

**交易流程**：
1. 自动获取实时价格
2. 验证持仓数量是否充足
3. 计算收益（先进先出成本法）
4. 回收资金到资金池，记录盈亏

**示例**：
```bash
# 卖出 50 股
python3 <skill根目录>/scripts/paper_trading.py sell "赛力斯" --qty 50

# 全部清仓
python3 <skill根目录>/scripts/paper_trading.py sell "赛力斯" --all

# 带备注
python3 <skill根目录>/scripts/paper_trading.py sell "中科曙光" --qty 100 --note "止损"
```

### 7. 生成报告

生成不同类型的分析报告。

```bash
python3 <skill根目录>/scripts/paper_trading.py report "股票名称" --type holdings   # 持仓报告
python3 <skill根目录>/scripts/paper_trading.py report "股票名称" --type profit     # 收益报告
python3 <skill根目录>/scripts/paper_trading.py report "股票名称" --type history    # 操作报告（同 operations）
```

**报告类型**：
- `holdings`：持仓报告（同 `holdings` 命令）
- `profit`：收益报告（详细盈亏分析）
- `history`：操作历史报告（同 `operations` 命令）

**收益报告示例**：
```markdown

- **总盈亏**: 📉 ¥-514.96
- **总收益率**: -0.52%

## 📋 单笔收益明细

| 时间 | 价格 | 数量 | 成交金额 | 成本 | 盈亏 | 收益率 |
|------|------|------|----------|------|------|--------|
```

## 常见使用场景

### 场景 1：完整交易流程

```bash
# 1. 初始化资金池
python3 <skill根目录>/scripts/paper_trading.py init "赛力斯" --capital 100000

# 2. 买入建仓
python3 <skill根目录>/scripts/paper_trading.py buy "赛力斯" --qty 500 --note "初始建仓"

# 3. 查看持仓
python3 <skill根目录>/scripts/paper_trading.py holdings "赛力斯"

# 4. 部分止盈
python3 <skill根目录>/scripts/paper_trading.py sell "赛力斯" --qty 200 --note "部分止盈"

# 5. 查看收益
python3 <skill根目录>/scripts/paper_trading.py report "赛力斯" --type profit
```

### 场景 2：按金额分段建仓

```bash
python3 <skill根目录>/scripts/paper_trading.py buy "赛力斯" --amount 30000 --note "第一段"
python3 <skill根目录>/scripts/paper_trading.py buy "赛力斯" --amount 20000 --note "第二段"
python3 <skill根目录>/scripts/paper_trading.py buy "赛力斯" --amount 10000 --note "第三段"
```

### 场景 3：快速检查多个股票

```bash
# 检查多个股票的资金池状态
for stock in 赛力斯 中科曙光 特斯拉; do
    echo "=== $stock ==="
    python3 <skill根目录>/scripts/paper_trading.py pool "$stock"
    echo ""
done
```

## 注意事项

1. **美股支持**：本系统仅支持 A股和港股，不支持美股代码
2. **资金验证**：买入操作前会严格检查资金池余额，不足时拒绝交易
3. **持股验证**：卖出操作前会验证持仓数量，超卖会报错
4. **实时价格**：每次交易操作都会自动获取最新价格
5. **数据持久化**：所有操作实时保存到 JSON 文件，数据不会丢失
6. **强制初始化**：使用 `--force` 会永久删除现有数据，请谨慎使用

## 数据文件说明

### holdings.json

存储当前持仓和资金池状态，包含：
- 股票基本信息
- 资金池状态（总资金、可用资金、占用资金）
- 持仓列表（每次买入/卖出的仓位记录）

### operations.json

存储所有历史操作记录，包含：
- 初始化操作
- 买入操作（价格、数量、金额、备注）
- 卖出操作（价格、数量、金额、成本、盈亏、备注）

两个文件共同维护，确保数据完整性：
- `holdings.json`：当前状态快照（频繁更新）
- `operations.json`：完整流水账（增量追加）

## 故障排查

### 问题 1：找不到股票代码

```bash
⚠️  未找到股票 '某某' 的 A股代码
```

**解决方法**：
1. 检查股票名称是否正确
2. 手动指定股票代码：`--code sh600000`
3. 使用其他查询工具确认股票代码

### 问题 2：资金不足

```bash
❌ 资金不足
   需要：¥10,000.00
   可用：¥5,000.00
   缺口：¥5,000.00
```

**解决方法**：
1. 减少买入数量或金额
2. 先卖出部分持仓释放资金
3. 使用 `--force` 重新初始化更大资金池

### 问题 3：无法获取实时价格

```bash
⚠️  StockPriceFetcher 不可用
或者
❌ 无法获取实时价格
```

**解决方法**：
1. 检查网络连接
2. 确认股票代码格式正确（sh/sz/hk 开头）
3. 尝试其他股票代码测试
