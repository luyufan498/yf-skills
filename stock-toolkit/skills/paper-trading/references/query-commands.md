## info 命令（推荐）

`info` 命令是合并展示命令，可以一次性查看资金池、持仓和收益信息，是最常用的查询命令。

### 命令语法

```bash
# 查看单个股票的完整信息
ptrade info "股票名称"

# 查看所有股票的概览
ptrade info
```

### 参数说明

| 参数 | 必需 | 说明 |
|------|------|------|
| `股票名称` | 否 | 指定股票名称，省略则显示所有 |

### 使用示例

```bash
# 查看单只股票完整信息
ptrade info "赛力斯"

# 查看所有股票概览
ptrade info
```

### 输出内容

info 命令会显示以下信息：

1. **股票基本信息**
   - 股票名称
   - 股票代码
   - 当前价格

2. **资金池状态**
   - 总资金（初始化金额）
   - 可用资金（剩余资金）
   - 占用资金（持仓占用）
   - 资金使用率

3. **持仓情况**
   - 持仓数量
   - 平均成本价
   - 当前市值
   - 浮动盈亏

4. **收益统计**
   - 实现盈亏（已卖出部分）
   - 浮动盈亏（持仓部分）
   - 总盈亏

### 何时使用 info 命令

- 交易前后需要快速了解股票完整状态
- 查看多个股票的概览信息
- 日常检查持仓和收益

## pool 资金池查询

查看指定股票的资金池状态，了解可用资金和资金使用情况。

### 命令语法

```bash
# 查看单个股票的资金池
ptrade pool "股票名称"

# 查看所有股票的资金池
ptrade pool
```

### 参数说明

| 参数 | 必需 | 说明 |
|------|------|------|
| `股票名称` | 否 | 指定股票名称，省略则显示所有 |

### 使用示例

```bash
# 查看单只股票资金池
ptrade pool "赛力斯"

# 查看所有股票资金池
ptrade pool
```

### 输出字段说明

| 字段 | 说明 | 计算方式 |
|------|------|---------|
| 总资金 | 初始化时的资金金额 | init 时设定的 --capital |
| 可用资金 | 当前可用于交易的资金 | 总资金 - 占用资金 |
| 占用资金 | 持仓股票占用的资金 | 持仓数量 × 平均价 |
| 资金使用率 | 资金占用比例 | (占用资金 / 总资金) × 100% |

### 资金池状态解读

| 使用率 | 状态 | 建议 |
|--------|------|------|
| 0% | 空仓 | 可建仓 |
| 0-30% | 轻仓 | 可加仓 |
| 30-50% | 中仓 | 谨慎操作 |
| 50-70% | 重仓 | 控制风险 |
| 70-100% | 满仓 | 只能卖，不能买 |

### 何时使用 pool 命令

- 买入前确认可用资金
- 评估资金利用效率
- 规划仓位分配

## holdings 持仓查询

查看指定股票的持仓情况，包括持仓数量、成本、市值等信息。

### 命令语法

```bash
# 查看单个股票的持仓
ptrade holdings "股票名称"

# 查看所有股票的持仓
ptrade holdings
```

### 参数说明

| 参数 | 必需 | 说明 |
|------|------|------|
| `股票名称` | 否 | 指定股票名称，省略则显示所有 |

### 使用示例

```bash
# 查看单只股票持仓
ptrade holdings "赛力斯"

# 查看所有股票持仓
ptrade holdings
```

### 输出字段说明

| 字段 | 说明 | 说明 |
|------|------|------|
| 股票名称 | 股票中文名称 | 如"赛力斯" |
| 股票代码 | 股票代码 | 如 sh603527 |
| 当前价格 | 最新股价 | 自动获取实时价格 |
| 持仓数量 | 当前持有股数 | 多次买入后的净持仓 |
| 平均成本 | 摊薄后的成本价 | 按FIFO原则计算 |
| 当前市值 | 持仓总价值 | 持仓数量 × 当前价格 |
| 浮动盈亏 | 库存盈亏 | 当前市值 - 持仓成本 |

### 成本计算说明

平均成本采用摊薄方式计算：

```
平均成本 = 总买入成本 / 总买入数量

示例：
- 第1批买入 100 股 @ 10 元 → 成本 1000 元
- 第2批买入 100 股 @ 12 元 → 成本 1200 元
- 第3批买入 100 股 @ 13 元 → 成本 1300 元

平均成本 = (1000 + 1200 + 1300) / 300 = 11.67 元/股
```

### 何时使用 holdings 命令

- 每日检查持仓状态
- 评估持仓成本
- 决定卖出时机

## operations 操作历史

查看指定股票的所有操作历史，包括初始化、买入、卖出操作。

### 命令语法

```bash
# 查看单个股票的操作历史
ptrade operations "股票名称"

# 查看所有股票的操作历史
ptrade operations
```

### 参数说明

| 参数 | 必需 | 说明 |
|------|------|------|
| `股票名称` | 否 | 指定股票名称，省略则显示所有 |

### 使用示例

```bash
# 查看单只股票操作历史
ptrade operations "赛力斯"

# 查看所有股票操作历史
ptrade operations
```

### 操作类型

操作历史中包含以下操作类型：

| 操作类型 | 说明 | 关键字段 |
|---------|------|---------|
| INIT | 初始化资金池 | 资金金额 |
| BUY | 买入股票 | 价格、数量、金额 |
| SELL | 卖出股票 | 价格、数量、金额、成本、盈亏 |

### 输出字段说明

| 字段 | 买入操作 | 卖出操作 |
|------|---------|---------|
| 时间 | 操作时间 | 操作时间 |
| 类型 | BUY | SELL |
| 价格 | 买入价 | 卖出价 |
| 数量 | 买入股数 | 卖出股数 |
| 金额 | 买入总额 | 卖出总额 |
| 成本 | - | 卖出成本（FIFO） |
| 盈亏 | - | 卖出盈亏（金额 - 成本） |
| 备注 | 用户备注 | 用户备注 |

### 何时使用 operations 命令

- 回顾交易决策
- 分析交易频率
- 核对交易记录
- 学习交易经验

## profit 收益报告

查看指定股票的收益报告，包括实现盈亏、浮动盈亏和总盈亏。

### 命令语法

```bash
# 查看单个股票的收益报告
ptrade profit "股票名称"

# 查看所有股票的收益报告
ptrade profit
```

### 参数说明

| 参数 | 必需 | 说明 |
|------|------|------|
| `股票名称` | 否 | 指定股票名称，省略则显示所有 |

### 使用示例

```bash
# 查看单只股票收益报告
ptrade profit "赛力斯"

# 查看所有股票收益报告
ptrade profit
```

### 输出字段说明

| 字段 | 说明 | 计算方式 |
|------|------|---------|
| 实现盈亏 | 已卖出部分的盈亏 | 所有 SELL 操作的盈亏总和 |
| 浮动盈亏 | 持仓部分的盈亏 | 当前市值 - 持仓成本 |
| 总盈亏 | 总盈亏 | 实现盈亏 + 浮动盈亏 |

### 盈亏计算详解

#### 实现盈亏

实现盈亏是已兑现的盈亏：

```python
实现盈亏 = Σ(每笔卖出金额 - 每笔卖出成本)

示例：
- 卖出1: 100股 @ 15元 = 1500元，成本 1000元 → 盈余 500元
- 卖出2: 50股 @ 14元 = 700元，成本 600元 → 盈余 100元

实现盈亏 = 500 + 100 = 600元
```

#### 浮动盈亏

浮动盈亏是未兑现的盈亏：

```python
浮动盈亏 = 当前市值 - 持仓成本

示例：
- 持仓 200 股，成本价 10元 → 持仓成本 2000元
- 当前价格 12元 → 当前市值 2400元

浮动盈亏 = 2400 - 2000 = 400元
```

#### 总盈亏

```python
总盈亏 = 实现盈亏 + 浮动盈亏
       = 600 + 400 = 1000元
```

### 何时使用 profit 命令

- 评估投资回报
- 计算收益率
- 对比不同股票表现
- 决定止盈止损

## list 列出账户

列出所有已初始化的股票账户。

### 命令语法

```bash
ptrade list
```

### 使用示例

```bash
ptrade list
```

### 输出内容

list 命令会显示所有账户的列表，包括：
- 股票名称
- 股票代码
- 资金池总金额
- 持仓数量
- 最后更新时间

### 何时使用 list 命令

- 查看所有账户概览
- 寻找特定账户
- 检查账户数量
