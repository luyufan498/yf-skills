# PPT 规划说明

## 概述

PPT 规划是生成演示文稿前的关键步骤，通过分析用户提供的材料，将其整理成有逻辑的章节结构，并生成详细的页面规划文档。

## 规划步骤

### 1. 材料分析

分析用户提供的文字、数据、图片等资料，识别核心要点和逻辑顺序。

### 2. 章节划分

根据材料内容划分章节，每个章节包含多个相关的内容页。

### 3. 页面分配

根据内容密度和重要性，合理分配每个章节的页面数量。

### 4. 生成规划文件

在工作目录（已由 SKILL 创建）下生成 `PPT规划.md` 文件，列出所有页面的详细信息。

### 5. 对PPT规划进行基本校验

启动一个子代理对整个PPT规划进行检查。

1. 检查页面是否有遗漏，包括首页、感谢页等必要页面
2. 检查目录页内容章节是否与实际正文内容对应
3. 检查整体结构是否符合本说明中关于页面规划的要求

## 页面规划结构

### ⚠️ 目录页（TOC）重要说明

**每个章节（Section）开头必须生成一份目录页。**

**目录页生成规则：**
- 每个章节**开头**必须生成一份目录页，高亮当前章节
- 目录页只列出章节标题，不列出具体内容页
- 不需要生成总目录，因为其与第一章的目录重复，保留第一章目录即可。
- 内容划分时，通常整体的章节数量不应超过 5 章，以免目录页过于拥挤影响排版效果。

### 章节结构示例

```markdown
封面页：01_home.html

第一章：背景概述
- 目录页：02_toc.html（高亮第一章）
- 内容页：03_项目背景介绍
- 内容页：04_核心问题分析

第二章：核心内容
- 目录页：05_toc.html（高亮第二章）
- 内容页：06_关键要点一
- ...

结束页：13_end.html
```

### 内容页分配原则

- 每个内容页聚焦一个核心主题
- 合理分配数据、图表、图片资源
- 控制每个内容页的信息密度
- 确保逻辑递进和连贯性
- 总页数控制在 8-20 页之间

### 页数规划建议

| 内容丰富度 | 建议页数 | 说明 |
|-----------|---------|------|
| 简介/概览 | 8-10 页 | 核心内容精简，聚焦重点 |
| 标准汇报 | 12-15 页 | 内容完整，有数据和案例 |
| 详细分析 | 16-20 页 | 深入展开，多角度展示 |

## PPT 规划文件模板

在工作目录下生成 `PPT规划.md` 文件，内容如下：

```markdown
# PPT 规划

## 基本信息

- **主题**: [PPT主题]
- **汇报人**: [姓名]
- **页数**: [总页数]
- **创建时间**: [YYYY-MM-DD]

## 页面规划

### 第01页：封面页 (01_home.html)
- **标题**: [主标题]
- **内容**: [简要描述]

### 第02页：章节目录 (02_toc.html)
- **标题**: [章节名]
- **内容**: [高亮当前章节]

### 第03页：[页面标题] (03_[描述].html)
- **标题**: [页面标题]
- **内容**: [简要内容概述]

### 第04页：[页面标题] (04_[描述].html)
- **标题**: [页面标题]
- **内容**: [简要内容概述]

### 第XX页：[页面标题] (XX_[描述].html)
- **标题**: [页面标题]
- **内容**: [简要内容概述]

### 第YY页：章节目录 (YY_toc.html)
- **标题**: [章节名]
- **内容**: [高亮当前章节]

### 第ZZ页：[页面标题] (ZZ_[描述].html)
- **标题**: [页面标题]
- **内容**: [简要内容概述]

### 第NN页：结束页 (NN_end.html)
- **说明**: 使用感谢页的模板即可。 
```

## 规划文件要求

- 按照页码顺序列出所有页面
- 每页包含文件名、标题、简要的内容要点
- 清晰标识章节结构
- 便于用户快速浏览和确认

## 使用说明

- 将模板中的占位符（如 `[PPT主题]`、`[页面标题]` 等）替换为实际内容
- 根据实际页数增删页面条目
- 确保页码与文件名一致，文件名格式为 `{页码:02d}_{简短描述}.html`
- 目录页必须按章节高亮显示当前章节

## 文件命名规范

格式：`{页码:02d}_{简短描述}.html`
- 封面页：`01_home.html`
- 目录页：`02_toc.html`、`03_toc.html` 等
- 内容页：`03_背景介绍.html`、`04_问题分析.html` 等
- 结束页：`16_end.html`

页码必须是 2 位数字，不足前面补 0。
