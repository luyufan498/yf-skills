## 投资组合概览

paper-trading 支持同时管理多只股票，每只股票都有独立的资金池和持仓，通过投资组合管理可以查看总体投资情况。

### 核心概念

| 概念 | 说明 |
|------|------|
| 独立资金池 | 每只股票的资金相互独立，不共享 |
| 独立持仓 | 每只股票的持仓独立计算成本和盈亏 |
| 统一管理 | 通过 portfolio 和 analyze 命令查看整体情况 |
| 性能统计 | 收益率、胜率、盈亏比等指标 |

### 适用场景

- 分散投资：同时持有多只股票分散风险
- 对比分析：比较不同股票的投资回报
- 资金分配：合理分配资金到不同股票
- 整体评估：评估整体投资组合的表现

## portfolio 命令

查看所有股票的投资组合报告，包括资金池、持仓、收益等综合信息。

### 命令语法

```bash
ptrade portfolio
```

### 使用示例

```bash
ptrade portfolio
```

### 输出内容

portfolio 命令会输出每位股票的详细信息和汇总统计：

#### 单只股票信息

| 字段 | 说明 |
|------|------|
| 股票名称 | 股票中文名称 |
| 股票代码 | 股票代码 |
| 总资金 | 初始化金额 |
| 可用资金 | 当前可用资金 |
| 占用资金 | 持仓占用资金 |
| 持仓数量 | 当前持有股数 |
| 当前市值 | 持仓总价值 |
| 实现盈亏 | 已卖出部分盈亏 |
| 浮动盈亏 | 持仓部分盈亏 |
| 总盈亏 | 总盈亏 |

#### 汇总统计

| 字段 | 说明 | 计算 |
|------|------|------|
| 股票数量 | 总股票数量 | Σ 各股票 |
| 总资金 | 所有资金池总和 | Σ 各股票总资金 |
| 总市值 | 所有持仓市值总和 | Σ 各股票当前市值 |
| 总盈亏 | 所有股票盈亏总和 | Σ 各股票总盈亏 |
| 总收益率 | 整体投资收益率 | (总盈亏 / 总资金) × 100% |

### 何时使用 portfolio 命令

- 查看所有股票的完整状态
- 评估整体投资回报
- 对比不同股票的表现
- 规划资金调整策略

## analyze 命令

查看所有股票的性能分析报告，包括收益率、胜率、盈亏比等关键指标。

### 命令语法

```bash
ptrade analyze
```

### 使用示例

```bash
ptrade analyze
```

### 输出内容

analyze 命令提供更专业的性能分析指标。

#### 收益率指标

| 指标 | 说明 | 计算 |
|------|------|------|
| 总资金 | 所有股票初始资金总和 | Σ 各股票 init 资金 |
| 总盈亏 | 所有股票盈亏总和 | 实现盈亏 + 浮动盈亏 |
| 收益率 | 整体投资收益率 | (总盈亏 / 总资金) × 100% |

#### 交易统计指标

| 指标 | 说明 | 计算 |
|------|------|------|
| 总交易次数 | 所有买卖操作总和 | Σ 各股票交易数 |
| 买入次数 | 买入操作总次数 | Σ 各股票 BUY 操作 |
| 卖出次数 | 卖出操作总次数 | Σ 各股票 SELL 操作 |

#### 盈亏统计指标

| 指标 | 说明 | 计算 |
|------|------|------|
| 盈利交易次数 | 盈余的卖出操作数 | SELL 中盈余次数 |
| 亏损交易次数 | 亏损的卖出操作数 | SELL 中亏损次数 |
| 胜率 | 盈利交易占比 | (盈利次数 / 总卖出次数) × 100% |
| 盈亏比 | 平均盈利 / 平均亏损 | 平均盈利 / 平均亏损（绝对值） |

#### 资金利用指标

| 指标 | 说明 | 计算 |
|------|------|------|
| 总可用资金 | 所有股票可用资金总和 | Σ 各股票可用资金 |
| 总占用资金 | 所有股票占用资金总和 | Σ 各股票占用资金 |
| 资金利用率 | 资金占用比例 | (总占用 / 总资金) × 100% |

### 关键指标解读

#### 收益率

- **正值**：整体投资盈利
- **负值**：整体投资亏损
- **目标**：通常期望收益率 > 10%

#### 胜率

- **高胜率（> 60%）**：策略稳定，但可能盈亏比不高
- **低胜率（< 40%）**：波动大，但可能有高盈亏比
- **理想**：胜率 50-60% + 盈亏比 > 1.5

#### 盈亏比

- **> 2**：优秀，盈利远大于亏损
- **1-2**：良好
- **< 1**：不佳，亏损大于盈利

#### 资金利用率

- **< 30%**：资金闲置，可考虑加仓
- **30-70%**：合理利用
- **> 70%**：资金紧张，慎用杠杆

### 何时使用 analyze 命令

- 定期评估投资策略有效性
- 了解交易系统的胜率和盈亏比
- 评估资金使用效率
- 发现投资组合的优劣势

## 多账户管理策略

### 资金分配原则

| 策略 | 资金分配 | 适用场景 |
|------|---------|---------|
| 均等分配 | 每只股票平均分配 | 分散投资，降低单一风险 |
| 价值加权 | 按市值比例分配 | 投资大盘股，降低波动 |
| 压缩集中 | 重点股票 40-60%，其余分散 | 有强信心标的时使用 |

### 示例资金分配

假设总资金 100,000 元：

```bash
# 均等分配：3 只股票
ptrade init "股票A" --capital 33333
ptrade init "股票B" --capital 33333
ptrade init "股票C" --capital 33334

# 压缩集中：1 只重点 + 2 只追踪
ptrade init "重点股票" --capital 50000
ptrade init "追踪股票A" --capital 25000
ptrade init "追踪股票B" --capital 25000
```

### 仓位管理建议

| 仓位比例 | 建议 | 风险等级 |
|---------|------|---------|
| 单股 < 30% | 可适当加仓 | 低 |
| 单股 30-50% | 保持当前仓位 | 中 |
| 单股 > 50% | 谨慎，考虑减仓 | 高 |
| 单股 > 80% | 高风险，建议分散 | 极高 |

### 多账户对比策略

通过 portfolio 命令对比不同股票的表现：

1. **查看收益率排名**：找出表现最好的股票
2. **分析胜率差异**：了解策略稳定性
3. **评估资金效率**：优化资金分配
4. **动态调整**：增持优质标的，减持弱势标的

## 收益率计算

### 单只股票收益率

```python
# 单股票收益率
收益率 = (总盈亏 / 总资金) × 100%

示例：
- 初始化金额：100,000 元
- 实现盈亏：+5,000 元
- 浮动盈亏：+3,000 元
- 总盈亏：+8,000 元

收益率 = (8000 / 100000) × 100% = 8%
```

### 整体投资组合收益率

```python
# 整体收益率
整体收益率 = (总组合盈亏 / 总组合资金) × 100%

示例：
- 所有股票总资金：300,000 元
- 股票A 盈亏：+5,000 元
- 股票B 盈亏：-2,000 元
- 股票C 盈亏：+8,000 元
- 总组合盈亏：+11,000 元

整体收益率 = (11000 / 300000) × 100% = 3.67%
```

### 年化收益率（估算）

```python
# 如果投资时间不足一年，可以按时间换算年化
年化收益率 = 收益率 / 投资天数 × 365

示例：
- 收益率：8%
- 投资天数：90 天

年化收益率 = (8% / 90) × 365 = 32.44%
```

**注意**：年化收益率为估算值，实际受多种因素影响。

## 投资组合优化

### 定期评估

建议定期（每周/每月）运行 `ptrade analyze` 评估：

```bash
# 每周分析
ptrade analyze

# 按时间追踪收益率变化
ptrade analyze > analysis_$(date +%Y-%m-%d).txt
```

### 调整策略

根据分析结果调整：

1. **高收益率股票**：
   - 考虑增加仓位（如 < 30%）
   - 评估是否适宜长期持有

2. **低胜率股票**：
   - 分析失败原因
   - 考虑减少仓位或清仓

3. **低盈亏比**：
   - 优化止盈止损策略
   - 减少亏损幅度

4. **高资金利用率**：
   - 考虑部分获利了结
   - 释放资金用于新机会
