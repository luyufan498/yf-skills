# 🏦 广发证券数据整合指南

## 📋 在股票分析中使用广发数据

注意：必须安装 gf-finance skill 工作流才能执行后续得流程，具体得操作参考gf-finance skill说明。

### 工作流程

```
1. 检查龙虎榜 → 判断主力资金态度 → 保存到文件
2. 获取财务数据 → 分析基本面 → 保存到文件
3. 查看 ETF 榜单 → 判断板块热度 → 保存到文件
4. 主 agent 读取保存的文件，整合到分析报告
```

---

## 🔍 龙虎榜数据使用

### 何时使用

- ✅ **交易日 16:00 后**: 获取当日龙虎榜,否则获取的是前一日
- ✅ **分析热门股票**: 判断机构/游资态度
- ✅ **异常波动分析**: 股票大涨/大跌后查看资金流向

### 数据解读

| 指标 | 含义 | 判断标准 |
|------|------|----------|
| **净买入** | 买入 - 卖出 | 正值为机构看好，负值为看空 |
| **知名营业部** | 机构/游资席位 | 知名席位买入=利好 |
| **上榜频率** | 近期上榜次数 | 频繁上榜=热门股票 |
| **上榜原因** | 涨幅/换手/偏离值 | 涨幅偏离=强势股 |

### 数据保存格式

获取龙虎榜数据后，保存为结构化 Markdown：

```markdown
# 龙虎榜数据报告

上榜原因: 日涨幅偏离值达 7%
买入前五: ¥2.3 亿
卖出前五: ¥1.8 亿
净买入: +¥5000 万
知名席位: 华泰证券上海分公司 (买入¥3000 万)
资金面判断: 🟢 机构看好，净买入明显
```

## 📊 财务基本面数据使用

### 何时使用

- ✅ **深度分析**: 了解公司基本面
- ✅ **估值判断**: PE/PB 历史分位
- ✅ **行业对比**: 判断行业地位

### 数据解读

| 维度 | 关键指标 | 健康标准 |
|------|----------|----------|
| **估值** | PE < 行业平均 | PB < 3 |
| **盈利** | ROE > 15% | 净利率 > 10% |
| **现金流** | 经营现金流为正 | 自由现金流为正 |
| **偿债** | 资产负债率 < 60% | 流动比率 > 1 |
| **成长** | 营收增长 > 20% | 利润增长 > 25% |

### 数据保存格式

获取财务数据后，保存为结构化 Markdown：

```markdown
# 财务基本面数据报告

估值指标:
  PE(TTM): 25.3 (行业前 30%, 45% 分位 - 合理)
  PB: 3.2 (行业前 40%, 52% 分位 - 合理)

盈利能力:
  ROE: 18.5% (行业前 20%)
  净利率: 12.3% (行业前 35%)

现金流:
  经营现金流: ¥120 亿 (健康)
  自由现金流: ¥85 亿 (充裕)

估值走势: 当前 PE 处于 1 年 45% 分位，估值合理

基本面判断: 🟢 行业龙头，盈利能力强，现金流健康
```

## 📈 ETF 榜单使用

### 何时使用

- ✅ **板块分析**: 判断所属行业热度
- ✅ **资金流向**: 查看 ETF 资金流入情况
- ✅ **指数估值**: 判断整体估值水平


### 数据保存格式

获取 ETF 数据后，保存为结构化 Markdown：

```markdown
# ETF 板块热度数据报告

相关 ETF: 新能源汽车 ETF (515030)
今日涨幅: +2.3% (涨幅榜第 5)
主力资金: +¥3.5 亿 (净流入)
溢价率: +0.5% (正常)
板块判断: 🟢 板块热度高，资金持续流入
```

## 🎯 完整分析流程示例

### 案例：分析赛力斯 (SH601127)

```bash
# 步骤 1: 检查龙虎榜
python3 scripts/lhb_analysis.py daily 20260310 sh | grep -A20 "赛力斯\|601127"

# 步骤 2: 获取财务数据
python3 scripts/quant_analysis.py basic SH601127
python3 scripts/quant_analysis.py trend SH601127 1y

# 步骤 3: 查看相关 ETF
python3 scripts/etf_rank.py search 汽车

# 步骤 4: 将数据整理并保存到文件
# 保存格式见下方"数据保存方式"章节
# 主 agent 会读取这些文件，整合到分析报告中
```

完成所有数据获取后，保存汇总数据：

```markdown

  新能源汽车 ETF: +2.1%，主力资金 +¥4 亿
```

## 💾 数据保存方式

### 保存路径规范

广发证券数据获取后，请参考下列命令将分析报告保存。

💾 路径说明：脚本内置环境变量处理机制，自动确定保存路径，请勿自行指定或手动写入文件。
📂 执行路径：当前说明文档上一级（`../`），请在执行命令前切换(cd)至此目录

### 保存命令

```bash
python3 scripts/analysis_manager.py save-data "<股票名称>" \
  --type gf-summary \
  --stdin << 'EOF'
# {股票名称} - 广发证券数据汇总

**数据获取时间**: {YYYY-MM-DD HH:mm:ss}
<!-- 注意：必须使用真实的系统时间，可通过 `date` 命令获取，不要使用项目配置中的 currentDate -->
---

## 📊 基本信息
- 股票代码、名称、所属行业

## 🔴 龙虎榜数据
[如当天没有上榜，标注：未上榜]

## 📈 财务基本面数据
[估值、盈利、现金流、偿债、成长五大维度]

## 📊 ETF 板块热度数据
[相关 ETF 表现和板块热度判断]

## 💡 综合判断
[基于广发数据的综合分析结论]

## 📚 数据来源
广发证券易淘金平台
EOF
```

## ⚠️ 注意事项

### 数据时效性

| 数据类型 | 更新时间 | 注意事项 |
|----------|----------|----------|
| 龙虎榜 | 交易日 16:00 后 | 非交易日获取上一日数据 |
| 财务数据 | 季报发布后 | 注意报告期 (12/13/14/15) |
| ETF 榜单 | 实时 | 交易时段每 3 秒更新 |

### 股票代码格式

- **上海**: `SH601127` (必须大写 SH 前缀)
- **深圳**: `SZ000776` (必须大写 SZ 前缀)
