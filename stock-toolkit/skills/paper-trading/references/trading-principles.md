## 核心交易纪律

### 1. 基于分析报告操作

**原则**：所有交易决策应基于分析报告和数据支撑，不得盲目操作。

**具体要求**：
- 交易前必须查看相关分析报告
- 明确分析报告中给出的投资建议（买入/卖出/持有）
- 交易备注中关联对应的分析报告来源
- 不能在无数据支持的情况下冲动交易

**示例**：
```bash
# 正确：基于分析报告的交易
ptrade buy "股票名称" --qty 100 --note "2026-04-07日分析报告：技术突破"

# 错误：无依据的交易
ptrade buy "股票名称" --qty 100
```

### 2. 严格执行止损止盈

**原则**：交易前设定明确的止损价和止盈价，到达价位坚决执行。

**具体要求**：
- 买入前必须设定止损价（通常建议亏损 5-10% 止损）
- 买入前必须设定止盈价（建议目标收益率 10-20%）
- 达到止损线立即止损，不抱有幻想
- 达到止盈线分批止盈，锁定利润

**示例**：
```bash
# 记录止损止盈计划
ptrade buy "股票名称" --qty 100 --note "止损价: 9元, 止盈价: 15元"

# 止盈时明确理由
ptrade sell "股票名称" --qty 50 --note "价格到达15元止盈目标，部分锁定利润"
```

### 3. 仓位控制原则

**原则**：合理分配资金，避免单只股票过度集中。

**仓位控制表**：

| 持仓比例 | 风险等级 | 建议 |
|---------|---------|------|
| < 30% | 低风险 | 可适当加仓 |
| 30-50% | 低风险 | 保持仓位，观察趋势 |
| 50-70% | 中风险 | 谨慎操作，控制加仓 |
| > 70% | 高风险 | 考虑减仓，降低风险 |

**初始建仓规则**：
- 空仓时首次建仓不超过总资金的 10-20%
- 单只股票最大持仓不超过总资金的 30%
- 重仓股（占比 > 50%）应特别谨慎，适时减仓

**示例**：
```bash
# 初始化时设定合理资金
ptrade init "股票名称" --capital 10000  # 如果总资金 10 万，占10%

# 查看资金利用率评估仓位
ptrade info "股票名称"

# 如果资金利用率超过 70%，考虑减仓
ptrade sell "股票名称" --qty 50 --note "风险控制：降低仓位"
```

### 4. 分批买卖原则

**原则**：大额资金建议分批交易，避免一次性梭哈。

**分批操作推荐**：

| 场景 | 策略 | 说明 |
|------|------|------|
| 首次建仓 | 2-3 批次 | 逐步建仓，降低成本 |
| 上涨突破 | 分批加仓 | 突破关键位后加仓 |
| 下跌破位 | 分批减仓 | 破位支撑时分批减仓 |
| 获利了结 | 分批卖出 | 分批锁定利润 |

**示例**：
```bash
# 首次建仓分批（3 批）
ptrade buy "股票名称" --qty 50 --note "建仓第1批"
ptrade buy "股票名称" --qty 50 --note "建仓第2批"
ptrade buy "股票名称" --qty 50 --note "建仓第3批，完成建仓"

# 分批止盈
ptrade sell "股票名称" --qty 33 --note "止盈第1批"
ptrade sell "股票名称" --qty 33 --note "止盈第2批"
ptrade sell "股票名称" --qty 34 --note "止盈第3批，清仓"
```

### 5. 避免频繁交易

**原则**：减少交易频率，降低手续费和心理成本。

**交易频率限制**：

| 时间单位 | 最大交易次数 | 说明 |
|---------|-------------|------|
| 每日 | 2 次 | 同一股票每日不超过 2 次交易 |
| 间隔 | 1 交易日 | 相同方向交易至少间隔 1 个交易日 |

**禁止操作**：
- 一日内多次买卖同一股票
- 频繁改变交易方向（先买后卖再买）
- 无交易逻辑的随机操作
- 短线追涨杀跌

**示例**：
```bash
# 合理：耐心等待
ptrade buy "股票名称" --qty 100 --note "建仓，等待趋势确认"
# （1-2 天后）
ptrade sell "股票名称" --qty 30 --note "部分止盈，降低风险"

# 错误：频繁交易
ptrade buy "股票名称" --qty 100 --note "建仓"
ptrade sell "股票名称" --qty 100 --note "已涨"
ptrade buy "股票名称" --qty 100 --note "又跌了"
```

## 禁止行为

### 1. 禁止盲目追涨杀跌

**具体表现**：
- 看到股票快速上涨立即追高买入
- 看到股票快速下跌恐慌杀跌卖出
- 不看基本面和技术面盲目操作
- 情绪化决策，冲动交易

**正确做法**：
```bash
# 错误：盲目追涨
ptrade buy "股票名称" --qty 100 --note "涨了这么快"

# 正确：基于分析
ptrade buy "股票名称" --qty 100 --note "分析报告：技术突破，趋势向上"
```

### 2. 禁止频繁操作

**具体表现**：
- 一日内对同一股票多次买卖
- 没有明确交易逻辑的随机操作
- 同一天内先买后卖再买
- 只要有波动就忍不住要交易

**影响**：
- 增加交易成本（手续费）
- 心理压力增大
- 心态失衡
- 容易出错

### 3. 禁止满仓梭哈

**具体表现**：
- 单只股票持仓占比 > 80%
- 一次性买入占用全部资金
- 不留任何现金应对机会
- 集中风险过高

**风险**：
- 单只股票下跌风险无法对冲
- 错失其他投资机会
- 心态容易失衡

**正确做法**：
```bash
# 错误：满仓梭哈
ptrade init "股票" --capital 100000
ptrade buy "股票" --amount 100000  # 全部买入

# 正确：保留现金
ptrade init "股票" --capital 50000  # 只用50%
ptrade buy "股票" --amount 25000   # 再用一半建仓
# 剩余现金用于：追加机会、对冲风险
```

### 4. 禁止摊平死扛

**具体表现**：
- 股票下跌亏损时不断加仓摊平成本
- 亏损达到止损线不止损，幻想反弹
- 一味死扛不承认错误
- 不及时止损，亏损扩大

**正确做法**：
```bash
# 错误：摊平死扛
ptrade buy "股票" --qty 100 --note "建仓"
# ... 股价下跌 10% ...
ptrade buy "股票" --qty 50 --note "摊平成本"
# ... 继续下跌 10% ...
ptrade buy "股票" --qty 50 --note "再摊平"

# 正确：及时止损
ptrade buy "股票" --qty 100 --note "建仓"
# ... 股价下跌 5% ...
ptrade sell "股票" --qty 100 --note "止损，亏损控制在5%"
```

## 交易决策流程

### 标准决策流程

```
1. 生成每日分析报告
   ├─ 技术分析图表
   ├─ 基本面分析
   └─ 市场环境评估
        ↓
2. 评估投资建议
   ├─ 明确建议方向（买入/卖出/持有）
   ├─ 确认风险等级
   └─ 理解建议理由
        ↓
3. 检查当前持仓状态
   ├─ ptrade holdings 查看持仓
   ├─ 评估浮动盈亏
   └─ 判断持仓比例
        ↓
4. 评估资金使用率
   ├─ ptrade pool 查看资金池
   ├─ 确认可用资金
   └─ 评估仓位控制
        ↓
5. 制定具体交易计划
   ├─ 确定交易数量/金额
   ├─ 制定分批策略
   ├─ 设定止损价和止盈价
   └─ 添加详细交易备注
        ↓
6. 执行交易
   ├─ ptrade buy / sell 执行
   ├─ 观察执行结果
   └─ 确认交易记录正确
        ↓
7. 事后记录和总结
   ├─ ptrade operations 回顾交易
   ├─ ptrade profit 评估盈亏
   └─ 总结经验教训
```

### 使用 ptrade 命令辅助决策

#### 步骤 1：查看当前状态

```bash
# 查看持仓和盈亏
ptrade info "股票名称"

# 查看资金池
ptrade pool "股票名称"

# 查看操作历史
ptrade operations "股票名称"

# 查看收益报告
ptrade profit "股票名称"
```

#### 步骤 2：获取市场数据

```bash
# 查看实时价格
ptrade fetch-price sh600000

# 查看K线分析
ptrade fetch-kline sh600000 --type day --count 30

# 获取相关新闻
ptrade fetch-news --source all --limit 10
```

#### 步骤 3：执行交易

```bash
# 分批建仓
ptrade buy "股票名称" --qty 50 --note "报告：技术突破，建仓第1批"
ptrade buy "股票名称" --qty 50 --note "趋势确认，建仓第2批"

# 设置止损止盈
ptrade sell "股票名称" --qty 33 --note "部分止盈，价格接近目标"
ptrade sell "股票名称" --qty 67 --note "清仓，趋势改变"
```

#### 步骤 4：后续跟踪

```bash
# 定期检查
ptrade holdings "股票名称"  # 检查持仓
ptrade fetch-price sh600000  # 最新价格
ptrade fetch-news -s cls    # 财联社快讯

# 整体评估
ptrade analyze  # 性能分析
```

## 仓位管理策略

### 1. 初始建仓策略

| 资金量 | 首次建仓比例 | 单股票最大持仓 | 说明 |
|--------|-------------|--------------|------|
| 小资金（<1万） | 20-30% | 50% | 资金少可以适当集中 |
| 中等资金（1-10万） | 10-20% | 40% | 平衡分散和集中 |
| 大资金（>10万） | 5-10% | 30% | 充分分散风险 |

### 2. 加仓策略

**川字金字塔加仓**：
- 底仓（30-40%）
- 第一次加仓（30-40%）→ 股价创新高
- 第二次加仓（20-30%）→ 再创新高

```bash
# 川字金字塔加仓示例
ptrade buy "股票" --qty 40 --note "底仓40股"
# ... 价格上涨 5% ...
ptrade buy "股票" --qty 30 --note "加仓30股，创新高"
# ... 价格再涨 5% ...
ptrade buy "股票" --qty 30 --note "继续加仓30股"
```

### 3. 减仓策略

**分批减仓**：
- 第一批减仓 30% → 止盈目标到
- 第二批减仓 30% → 趋势转弱
- 第三批清仓 → 趋势破坏

```bash
# 分批减仓示例
ptrade sell "股票" --qty 30 --note "价格到达目标，部分止盈"
ptrade sell "股票" --qty 30 --note "趋势转弱，继续减仓"
ptrade sell "股票" --qty 40 --note "趋势破坏，清仓离场"
```

## 止盈止损策略

### 止损策略

| 情况 | 止损标准 | 说明 |
|------|---------|------|
| 技术破位 | 跌破关键支撑位 | 技术止损 |
| 金额止损 | 亏损 5-10% | 金钱止损 |
| 时间止损 | 持有 X 天未涨 | 避免资金占用 |
| 消息止损 | 重大利空消息 | 消息止损 |

```bash
# 技术止损：跌破支撑
ptrade sell "股票" --qty 100 --note "技术止损：跌破20日均线支撑"

# 金额止损：亏损8%
ptrade sell "股票" --qty 100 --note "金额止损：亏损已超8%，保本金"

# 时间止损：持有15天无起色
ptrade sell "股票" --qty 100 --note "时间止损：持有15天未涨，释放资金"
```

### 止盈策略

| 情况 | 止盈标准 | 说明 |
|------|---------|------|
| 目标止盈 | 达到预期收益率 | 按计划执行 |
| 技术止盈 | 技术指标顶背离 | 技术信号 |
| 消息止盈 | 重大利好兑现 | 利好兑现 |
| 市场止盈 | 市场环境转弱 | 系统性风险 |

```bash
# 目标止盈：收益20%
ptrade sell "股票" --qty 50 --note "目标止盈：收益率已20%，部分锁定"

# 技术止盈：顶背离
ptrade sell "股票" --qty 50 --note "技术止盈：MACD顶背离信号"

# 市场止盈：大盘暴跌
ptrade sell "股票" --qty 100 --note "市场止盈：大盘暴跌，系统性风险释放"
```

## 交易频率控制

### 交易成本考量

每次交易都有隐形成本：
- **手续费**：券商收取（虽然模拟盘免费，但实际交易需要）
- **时间成本**：盯盘、研究、决策
- **心理成本**：频繁交易导致心态失衡
- **机会成本**：资金被占用

### 建议频率

| 投资风格 | 交易频率 | 特点 |
|---------|---------|------|
| 长线投资 | 月/季度级 | 把握大趋势，减少交易 |
| 波段操作 | 周级别 | 抓住波段机会 |
| 短线交易 | 日级别 | 需要大量时间和精力 |

### 频率检查

定期检查交易频率：

```bash
# 查看操作历史
ptrade operations "股票名称"

# 统计交易次数（一周内）
# 如果超过10次交易 → 就太频繁了
```

## 情绪管理

### 克服贪婪

- 不要预期不切实际的收益率
- 到达止盈目标要执行，不要贪更多
- 避免频繁交易，耐心等待机会

### 克服恐惧

- 亏损并不可怕，可怕的是不止损
- 不要因为一次亏损就不敢交易
- 严格执行计划，克服心理压力

### 保持平常心

- 模拟交易是学习，不是赌博
- 关注长期结果，不要纠结单次得失
- 交易前充分研究，提高成功率

### 使用日记记录

```bash
# 在交易备注中记录心态
ptrade buy "股票" --qty 100 --note "心态冷静，分析充分"
ptrade sell "股票" --qty 50 --note "获利了结，心态稳定"
```

## 总结

### 交易黄金法则

| 法则 | 要点 |
|------|------|
| 首次不大仓位 | 初始建仓 10-20% |
| 止损要坚决 | 不抱幻想，执行计划 |
| 止盈要理性 | 分批止盈，锁定利润 |
| 不追涨杀跌 | 基于分析，避免情绪化 |
| 不频繁交易 | 减少不必要的交易 |
| 不满仓梭哈 | 保留现金，控制风险 |
| 不摊平死扛 | 及时止损，保本金 |

### 关键指标监控

通过 `ptrade analyze` 定期监控：

| 指标 | 健康值 | 需注意 | 危险 |
|------|--------|--------|------|
| 收益率 | > 10% | 5-10% | < 0 |
| 胜率 | > 50% | 40-50% | < 40% |
| 盈亏比 | > 1.5 | 1-1.5 | < 1 |
| 资金利用率 | 30-70% | 70-90% | > 90% |
