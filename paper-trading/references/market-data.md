## 市场数据功能概览

paper-trading 提供完整的市场数据查询功能，支持 A股、港股、美股的价格、K线、代码搜索和新闻获取。

### 支持的市场

| 市场 | 代码格式 | 示例 |
|------|---------|------|
| A股 | sh[6位] 或 sz[6位] | sh600000, sz000001 |
| 港股 | hk[5位] | hk00700, hk09988 |
| 美股 | 直接代码 | AAPL, GOOGL, TSLA |

### 数据来源

| 数据类型 | 数据源 | 说明 |
|---------|-------|------|
| A股价格 | 腾讯财经 | 实时或准实时价格 |
| 港股价格 | 腾讯财经 | 实时或准实时价格 |
| 美股价格 | 新浪财经 | 实时或准实时价格 |
| K线数据 | 腾讯财经/Yahoo Finance | 历史K线数据 |
| 股票搜索 | 新浪财经综合 | A股、港股、美股代码 |
| 市场新闻 | 财联社/新浪/TradingView | 多源新闻聚合 |

## fetch-price 实时价格查询

查询指定股票的实时价格信息。

### 命令语法

```bash
ptrade fetch-price <股票代码> [--format 格式]
```

### 参数说明

| 参数 | 必需 | 说明 |
|------|------|------|
| `<股票代码>` | 是 | 股票代码（sh600000, hk00700, AAPL 等）|
| `--format` / `-f` | 否 | 输出格式：pretty（默认）或 json |

### 使用示例

```bash
# 查询A股价格（人可读格式）
ptrade fetch-price sh600000

# 查询港股价格
ptrade fetch-price hk00700

# 查询美股价格
ptrade fetch-price AAPL

# 输出 JSON 格式
ptrade fetch-price sh600000 --format json
```

### 输出内容

#### pretty 格式

```
股票名称: 浦发银行
股票代码: sh600000
当前价格: 12.50
涨跌幅: +1.20%
更新时间: 2026-04-07 10:30:00
```

#### json 格式

```json
{
  "code": "sh600000",
  "name": "浦发银行",
  "price": 12.50,
  "change": "+1.20%",
  "update_time": "2026-04-07T10:30:00",
  "market": "A股"
}
```

### 何时使用 fetch-price

- 交易前确认最新价格
- 监控持仓股票价格
- 价格走势分析
- 对比多个股票价格

## fetch-kline K线数据查询

查询指定股票的历史K线数据，支持多种时间周期。

### 命令语法

```bash
ptrade fetch-kline <股票代码> [--type 类型] [--count 数量] [--format 格式]
```

### 参数说明

| 参数 | 必需 | 默认值 | 说明 |
|------|------|--------|------|
| `<股票代码>` | 是 | - | 股票代码 |
| `--type` / `-t` | 否 | day | K线类型：day, week, month, 5min, 10min, 15min, 30min, 60min |
| `--count` / `-n` | 否 | 30 | 获取K线数量 |
| `--format` / `-f` | 否 | pretty | 输出格式：pretty 或 json |

### K线类型说明

| K线类型 | 说明 | 周期 | 适用场景 |
|---------|------|------|---------|
| day | 日K | 1天 | 长线分析 |
| week | 周K | 1周 | 波段分析 |
| month | 月K | 1月 | 趋势判断 |
| 5min | 5分钟K | 5分钟 | 日内交易 |
| 10min | 10分钟K | 10分钟 | 短线交易 |
| 15min | 15分钟K | 15分钟 | 短线分析 |
| 30min | 30分钟K | 30分钟 | 中短线 |
| 60min | 60分钟K | 60分钟 | 中线 |

### 使用示例

```bash
# 查询日K线（默认）
ptrade fetch-kline sh600000

# 查询周K线
ptrade fetch-kline sh600000 --type week

# 查询5分钟K线
ptrade fetch-kline sh600000 --type 5min

# 查询指定数量（50根）
ptrade fetch-kline sh600000 --type day --count 50

# 输出 JSON 格式
ptrade fetch-kline sh600000 --type day --format json
```

### 输出内容

#### pretty 格式

```
股票代码: sh600000
股票名称: 浦发银行
K线类型: 日K
更新时间: 2026-04-07 15:00:00

日期       开盘     最高     最低     收盘     成交量
2026-04-07  12.30   12.60   12.20   12.50    100000
2026-04-06  12.10   12.40   12.05   12.30    90000
...
```

#### json 格式

```json
{
  "code": "sh600000",
  "name": "浦发银行",
  "type": "day",
  "update_time": "2026-04-07T15:00:00",
  "kline": [
    {
      "date": "2026-04-07",
      "open": 12.30,
      "high": 12.60,
      "low": 12.20,
      "close": 12.50,
      "volume": 100000
    },
    ...
  ]
}
```

### K线数据分析

K线数据可以用于：

- **趋势分析**：通过价格走势判断趋势
- **支撑压力位**：找出历史高低点
- **成交量分析**：量价关系判断
- **技术指标**：结合技术指标进行分析
- **回测策略**：历史数据验证交易策略

### 何时使用 fetch-kline

- 交易前分析趋势
- 找出支撑压力位
- 制定交易计划
- 验证技术分析
- 回测交易策略

## search 股票搜索

搜索股票代码，支持A股、港股、美股综合搜索。

### 命令语法

```bash
ptrade search <关键词> [--limit 数量] [--format 格式]
```

### 参数说明

| 参数 | 必需 | 默认值 | 说明 |
|------|------|--------|------|
| `<关键词>` | 是 | - | 搜索关键词（股票名称或部分代码）|
| `--limit` / `-n` | 否 | 5 | 返回结果数量 |
| `--format` / `-f` | 否 | pretty | 输出格式：pretty 或 json |

### 使用示例

```bash
# 搜索股票
ptrade search 茅台

# 搜索并返回更多结果
ptrade search 茅台 --limit 10

# 输出 JSON 格式
ptrade search 茅台 --format json
```

### 输出内容

#### pretty 格式

```
搜索关键词: 茅台
找到 3 个结果:

1. 股票名称: 贵州茅台
   股票代码: sh600519
   市场: A股

2. 股票名称: 茅台股份
   股票代码: sh600526
   市场: A股

3. 股票名称: 茅台科技
   股票代码: sz000789
   市场: A股
```

#### json 格式

```json
{
  "keyword": "茅台",
  "count": 3,
  "results": [
    {
      "name": "贵州茅台",
      "code": "sh600519",
      "market": "A股"
    },
    ...
  ]
}
```

### 搜索策略

支持以下搜索方式：

| 搜索方式 | 示例 | 说明 |
|---------|------|------|
| 股票名称搜索 | "茅台" | 匹配股票名称包含关键词 |
| 部分代码搜索 | "600519" | 匹配股票代码的前几位 |
| 公司简称搜索 | "腾讯" | 匹配公司简称 |

### 热门股票库

对于港股和美股，虽然 API 搜索功能有限，但系统内置了热门股票库，可以直接获取：

**港股热门股票**：腾讯(00700)、阿里巴巴(09988)、美团(03690)等

**美股热门股票**：苹果(AAPL)、谷歌(GOOGL)、微软(MSFT)、特斯拉(TSLA)等

### 何时使用 search

- 不确定股票代码时
- 寻找特定公司股票
- 发现相似股票
- 了解市场板块相关股票

## fetch-news 市场新闻

获取多个来源的市场新闻，支持财联社、新浪财经、TradingView。

### 命令语法

```bash
ptrade fetch-news [--source 来源] [--limit 数量] [--format 格式]
```

### 参数说明

| 参数 | 必需 | 默认值 | 说明 |
|------|------|--------|------|
| `--source` / `-s` | 否 | all | 新闻来源：all（全部）, cls（财联社）, sina（新浪财经）, tv（TradingView）|
| `--limit` / `-n` | 否 | 10 | 新闻数量 |
| `--format` / `-f` | 否 | pretty | 输出格式：pretty 或 json |

### 新闻来源说明

| 新闻源 | 来源类型 | 特点 | 时效性 |
|--------|---------|------|--------|
| cls (财联社) | 财联社电报 | 快讯、重要市场资讯 | 极快 |
| sina (新浪财经) | 新浪财经直播 | 综合新闻、焦点新闻 | 快 |
| tv (TradingView) | TradingView 外媒 | 国际财经新闻 | 中等 |
| all (全部) | 所有来源合并 | 综合所有新闻源 | - |

### 使用示例

```bash
# 获取所有来源的新闻（默认）
ptrade fetch-news

# 只获取财联社新闻
ptrade fetch-news --source cls

# 只获取新浪财经新闻
ptrade fetch-news --source sina

# 只获取 TradingView 新闻
ptrade fetch-news --source tv

# 获取更多新闻
ptrade fetch-news --source all --limit 20

# 输出 JSON 格式
ptrade fetch-news --source all --format json
```

### 输出内容

#### pretty 格式

```
新闻来源: 财联社电报
获取时间: 2026-04-07 10:30:00

【重要】市场快讯 (10:25)
标题: 央行开展1000亿元7天期逆回购操作
内容: 为维护银行体系流动性合理充裕...
标签: [宏观, 逆回购]
链接: https://www.cls.cn/...
```

#### json 格式

```json
{
  "source": "cls",
  "fetch_time": "2026-04-07T10:30:00",
  "count": 10,
  "news": [
    {
      "title": "央行开展1000亿元7天期逆回购操作",
      "content": "为维护银行体系流动性合理充裕...",
      "time": "10:25",
      "date": "2026-04-07",
      "url": "https://www.cls.cn/...",
      "source_type": "cls",
      "is_red": true,
      "tags": ["宏观", "逆回购"]
    },
    ...
  ]
}
```

### 新闻字段说明

| 字段 | 说明 |
|------|------|
| title | 新闻标题 |
| content | 新闻内容 |
| time | 发布时间（时分秒）|
| date | 发布日期 |
| url | 新闻链接 |
| source_type | 新闻来源（cls/sina/tradingview）|
| is_red | 是否为重要新闻 |
| tags | 新闻标签 |

### 新闻获取特性

新闻获取器包含以下生产级特性：

- **重试机制**：网络失败自动重试（最多2次）
- **速率限制**：防止频繁触发 API 限制（默认0.5秒延迟）
- **超时控制**：可配置超时时间（默认10秒）
- **错误处理**：友好的错误提示
- **时区处理**：正确处理时间戳和日期

### 何时使用 fetch-news

- 每日开盘前了解市场动态
- 监控重要宏观新闻
- 关注特定股票相关新闻
- 获取外媒观点（TradingView）
